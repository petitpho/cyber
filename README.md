TPs : les bases de la Cybersécurité :


1. TP DHCP Starvation : Simulation d’une attaque DHCP Starvation et sécurisation du réseau

Dans ce TP, on simule une attaque réseau appelée DHCP Starvation. 
Cette attaque consiste à saturer le serveur DHCP avec un grand nombre de requêtes afin 
d'épuiser le pool d'adresses IP disponibles.
L’objectif est d’observer l’impact de cette attaque sur le réseau et de mettre en place un 
mécanisme de protection sur le switch.

Logiciel utilisé : Cisco Packet Tracer



2. TP sur la sécurisation des réseaux : Sécurisation avancée du réseau (VLAN, DHCP Snooping, Port 
Security et ACL)

Dans ce TP, on met en place plusieurs mécanismes de sécurité afin de protéger 
l'infrastructure réseau contre les attaques de type DoS.

Logiciel utilisé : Cisco Packet Tracer
